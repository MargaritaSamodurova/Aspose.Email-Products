<p>
 Aspose.Email для Android через Java позволяет разработчикам программного обеспечения создавать приложения для Android, имеющие возможность управлять почтовыми сообщениями и манипулировать ими, не запутываясь в сложностях реализации формата файлов Microsoft Outlook. Библиотека электронной почты Android помогает разработчикам управлять форматами почтовых файлов из приложений Android. API не имеет никаких зависимостей, кроме Android Runtime, поэтому для работы ему не нужны никакие другие API или программное обеспечение.
</p>

<p>
 Aspose.Email для Android через Java предоставляет инструменты, необходимые для создания, чтения и управления файлами Outlook MSG, PST, EML и MHT из приложения Android. Он предоставляет возможность манипулировать заголовками сообщений, содержимым, вложениями и свойствами Mapi в соответствии со спецификациями приложения.
</p>
