<p>
 Aspose.Email für Android über Java versetzt Softwareentwickler in die Lage, Android-Anwendungen zu entwerfen, die E-Mail-Nachrichten verwalten und manipulieren können, ohne sich in der Komplexität der Implementierung des Microsoft Outlook-Dateiformats zu verlieren. Die Android-E-Mail-Bibliothek hilft den Entwicklern, E-Mail-Dateiformate aus Android-Anwendungen heraus zu manipulieren. API hat keine anderen Abhängigkeiten als Android Runtime, daher benötigt es keine anderen APIs oder Software, um zu funktionieren.
</p>

<p>
 Aspose.Email für Android über Java bietet Ihnen die Tools, die Sie zum Erstellen, Lesen und Bearbeiten von Outlook MSG-, PST-, EML- und MHT-Dateien aus einer Android-Anwendung heraus benötigen. Es bietet die Möglichkeit, Nachrichtenheader, Inhalte, Anhänge und Mapi-Eigenschaften gemäß den Anwendungsspezifikationen zu bearbeiten.
</p>
