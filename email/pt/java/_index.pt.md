<p>
 Aspose.Email para Java permite que os desenvolvedores se concentrem na implementação de sua própria lógica de negócios para gerenciar e-mails sem ter que saber sobre especificações de formato ou programação de rede.
</p>

<p>
 Aspose.Email para Java pode criar e-mails do Outlook, analisar e-mails ou converter formatos de mensagem como MSG, EML, MBOX, PST, OST e MHT de aplicativos Java. A API pode manipular o conte&#250;do da mensagem, anexos, cabe&#231;alhos e propriedades MAPI, bem como a capacidade de enviar e receber e-mails por meio de v&#225;rios protocolos populares. Ele tamb&#233;m fornece a capacidade de gerenciar documentos de armazenamento de mensagens, como arquivos de armazenamento pessoal PST e arquivos de armazenamento off-line OST. Voc&#234; pode listar e-mails, extrair e-mails, adicionar ou remover mensagens de arquivos de armazenamento de acordo com as especifica&#231;&#245;es do seu aplicativo. A biblioteca Java Email permite trabalhar com o arquivo de contatos do Outlook (VCF).
</p>
