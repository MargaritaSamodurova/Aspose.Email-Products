<p>
 Aspose.Email pour Android via Java permet aux développeurs de logiciels de concevoir des applications Android capables de gérer et de manipuler des messages électroniques sans se perdre dans les complexités de la mise en œuvre du format de fichier Microsoft Outlook. La bibliothèque de messagerie Android aide les développeurs à manipuler les formats de fichiers de messagerie à partir des applications Android. L'API n'a aucune dépendance autre qu'Android Runtime, elle n'a donc pas besoin d'autres API ou logiciels pour fonctionner.
</p>

<p>
 Aspose.Email pour Android via Java vous donne les outils dont vous avez besoin pour créer, lire et manipuler les fichiers Outlook MSG, PST, EML et MHT à partir d'une application Android. Il offre la possibilité de manipuler les en-têtes de message, le contenu, les pièces jointes et les propriétés Mapi conformément aux spécifications de l'application.
</p>
